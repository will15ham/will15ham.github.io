I decided to use an API that returns info on the requests it recieves. Here we can see an example of generating a random user-agent and sending it to the API. Also the API has another endpoint that returns IP address. This can be useful when trying to build a request client that needs to match the browser, this api would allow you to see the request method, headers you are sending, body, and even some of the TLS specs.
